## Short description
Minimal java agent example.

## Build
### Prerequisites 
* Gradle > 4.2.1
* JDK8

### Build
* Clone repository
* Run gradle jar

### Usage
To use the instrumentation within a java application, you have to add the jar file by jvm parameter *-javaagent*, e.g.
-javaagent:<fullPathToExampleAgentJarFile>
